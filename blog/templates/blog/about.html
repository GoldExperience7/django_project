About

A web application built using Django featuring a dynamic post feed which utilizes sentiment analysis. A user can register with an account to make/view posts or provide an amazon (.in) reviews page URL to be scraped for posts automatically . The feed features two sorting mechanisms which order current posts in the feed by the total sentiment (polarity) value of a post in ascending or descending. The sorting is performed internally with the help of priority queues. Posts are divided into five ranks based on the calculated value. The web app also features an analytics page that displays tabulated and graphical information about the posts.
